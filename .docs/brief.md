# MoneyApp - Ganhe Dinheiro Online
- One-line positioning: App gamificado onde usuários completam tarefas para ganhar moedas e sacar via PIX
- Target users: Brasileiros buscando renda extra online, iniciantes em apps de recompensa
- Core features:
  1. Dashboard com saldo de moedas e histórico de ganhos
  2. Lista de tarefas (assistir vídeo, preencher formulário, indicar amigos) para ganhar moedas
  3. Tela de saque com conversão moedas → R$ e integração Firebase
  4. Guia passo a passo de como configurar Firebase e pagar usuários
  5. Ranking de usuários com mais ganhos
- Device strategy: adaptive
- Design style: moderno, verde/dourado, gamificado com badges e progresso
- Technical constraints: sem banco real, estado local com zustand, guia Firebase educativo
- Nova Agent: not needed
- Completed: brief.md
